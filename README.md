# DeliMone - デリバリー収益管理アプリ

## 概要

DeliMoneは、フードデリバリー配達員向けの収益管理・可視化アプリケーションです。複数のデリバリーサービスの収益を一元管理し、複雑な支払いサイクルに対応した正確な入金予定日の把握をサポートします。

## 主な機能

- **複数サービスの収益管理**: UberEatsや出前館など、複数のサービスの売上をまとめて記録。
- **詳細な支払いサイクル設定**: 各社の異なる支払いサイクル（週払い、月2回など）に柔軟に対応。
- **正確な入金日予測**: 祝日を考慮した入金予定日を自動で計算。
- **日々の稼働記録**: 稼働時間、休憩、配達ごとの売上、経費を簡単に記録。
- **キャッシュフローの可視化**: アプリ内カレンダーや入金予定リストで、いつ、いくら入金されるのかを直感的に把握。
- **パフォーマンス分析**: 売上推移グラフや時給計算により、自身の稼働効率を分析。
- **PWA対応**: スマートフォンのホーム画面にインストールし、ネイティブアプリのように利用可能。
