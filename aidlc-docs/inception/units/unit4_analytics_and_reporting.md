# Unit 4: 分析とレポート

この作業ユニットは、記録されたデータを分析し、配達員が自身の業績を理解しやすくするための可視化機能に焦点を当てています。

## User Stories

### Story 4.1: 売上推移グラフ
**As a** 配達員,
**I want to** 「月ごと」「週ごと」の売上推移と内訳グラフを表示する,
**so that** 自分の収益トレンドやどのサービスが稼げているかを視覚的に把握したい。

#### Acceptance Criteria
- [ ] 期間（月ごと／週ごと）を切り替えて売上推移を確認できる。
- [ ] グラフは棒グラフや折れ線グラフで表示される。
- [ ] 各期間の総売上に加えて、サービスごとの売上内訳が色分けなどで表示される。
- [ ] 記録した経費もグラフ上で確認できる（例：売上から経費を引いた純利益表示）。

### Story 4.2: 時給計算
**As a** 配達員,
**I want to** 稼働時間と売上に基づき、時給（日次・期間平均）を算出して表示する,
**so that** 自分の稼働効率を客観的な数値で評価したい。

#### Acceptance Criteria
- [ ] その日の実稼働時間と売上合計から、日次の時給が計算され表示される。
- [ ] 指定した期間（週ごと／月ごと）の平均時給が計算され表示される。
- [ ] 時給は「売上 ÷ 実稼働時間」で計算される。
